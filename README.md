# ssufid-sites

[ssufid](https://github.com/yourssu/ssufid)에서 크롤링한 웹사이트 데이터를 저장 및 배포하는 레포지토리

## Format

|배포 타입|url 형식|
|:-:|:-:|
|RSS|`https://ssufid.yourssu.com/<IDENTIFIER>/rss.xml`|
|JSON|`https://ssufid.yourssu.com/<IDENTIFIER>/data.json`|
